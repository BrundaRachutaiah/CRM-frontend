# 📊 CRM Frontend

A modern and responsive CRM frontend application that allows users to manage leads, add comments, track sales agents, and view reports.

Built using **React.js (Vite)** with a clean UI and fast performance.

---

## 🚀 Features

### 📌 Leads Management

* Create new leads
* View all leads
* Update lead details
* Delete leads

### 💬 Comments

* Add comments to leads
* View comments for each lead

### 🏷️ Tags

* Assign tags to leads
* View available tags

### 👨‍💼 Sales Agents

* View sales agents
* Assign agents to leads

### 📊 Reports Dashboard

* Last week performance
* Sales pipeline report
* Closed deals by agent

---

## 🛠️ Tech Stack

* React.js (Vite)
* React Router
* Axios
* CSS / Tailwind

---

## ⚡ Installation & Setup

```bash
git clone https://github.com/BrundaRachutaiah/CRM-frontend.git
cd CRM-frontend-main
npm install
```

---

## ▶️ Run the App

```bash
npm run dev
```

App runs at:

```
http://localhost:5173
```

---

## 🔗 Backend API

Make sure backend is running:

```
http://localhost:5000
```
